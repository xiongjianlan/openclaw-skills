# 似水流年，此情可待成追忆

一部长篇小说，讲述先天性心脏病女孩林念安与建筑师沈亦辰之间刻骨铭心的爱情故事。

## 故事简介

林念安，一个患有先天性心脏病的女孩，在生命即将走向尽头时选择独自远行，在江南水乡乌镇遇到了建筑师沈亦辰。两个各自背负着秘密与伤痛的灵魂相遇，却在现实的重重阻碍下，面临着爱与放手的选择。

## 人物设定

- **林念安**：先天性心脏病患者，温柔坚强，热爱绘画
- **沈亦辰**：建筑师，家族债务缠身，隐忍深情
